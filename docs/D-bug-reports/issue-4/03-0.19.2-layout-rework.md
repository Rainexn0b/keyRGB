# Issue 4 — 0.19.2 layout rework and scope narrowing

GitHub issue: https://github.com/Rainexn0b/keyRGB/issues/4

Date: early April 2026

Reporter findings on `0.19.2`:

1. Layout selection existed and was convenient.
2. The per-key editor backdrop could not be disabled fully.
3. Wrong-key lighting was resolved.
4. Multi-LED logical keys still only drove the last mapped LED.
5. Software blinking remained present.
6. Hardware speed still required further investigation.

Maintainer response:

- The issue had narrowed to layout/backdrop parity, multi-cell keymap support,
  and `ite8910` runtime behavior.
- A follow-up `0.20.0` iteration was planned with speed-specific support logging.
