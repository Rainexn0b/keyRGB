Single-line release procedure:

1) Run tests
	pytest -q

2) Bump versions
	- pyproject.toml: [project].version
	- packaging/rpm/keyrgb.spec: Version + %changelog (if you ship RPM)
	- packaging/rpm/README.md: VERSION example (optional)

3) Commit + tag
	git add -A
	git commit -m "Release vX.Y.Z"
	git tag -a vX.Y.Z -m "release vX.Y.Z"

4) Build artifacts (sdist + wheel)
	python3 -m pip install --user --upgrade build
	python3 -m build

5) Push (when ready)
	git push
	git push --tags
