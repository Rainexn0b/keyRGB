Single-line release procedure:

1) Run tests
	pytest -q

2) Bump versions
	- pyproject.toml: [project].version
	- CHANGELOG.md: add a release entry

3) Commit + tag
	git add -A
	git commit -m "Release vX.Y.Z"
	git tag -a vX.Y.Z -m "release vX.Y.Z"

4) Build artifacts (sdist + wheel)
	python -m buildpython --profile=release

5) Push (when ready)
	git push
	git push --tags
